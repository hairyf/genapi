# Playground

Try genapi in the browser: paste Swagger/OpenAPI JSON and pick a preset and mode to see generated code.

- **Presets**: axios, fetch, ky, got, ofetch, uni, TanStack Query (React/Vue)
- **Modes**: TypeScript, JavaScript, Schema (fetch/ofetch only), or React/Vue for TanStack Query

Generated files (e.g. `index.ts`, `index.type.ts`, `index.api.ts` for query presets) appear in tabs. Edit the JSON to update the output.
